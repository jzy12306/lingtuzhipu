# 规则优化文档

## 1. 核心规则

### 1.1 语言规范
- **始终使用简体中文**进行所有交流，避免使用繁体字或英文（专业术语或专有名称除外）（除非必要且已提供中文解释）
- 保持回答简洁明了，避免冗长的解释

### 1.2 问题修改流程(在单纯的查看文件时不必遵循规则)
1. **定位错误**：精确指出问题出现的位置和上下文
2. **分析原因**：深入分析问题产生的根本原因
3. **提供方案**：给出具体、可操作的解决方案
4. **验证效果**：确保解决方案能有效解决问题

### 1.3 回答格式要求
每次回答后必须包含：
- **功能实现说明**：明确说明已实现的功能点
- **技术建议**：推荐相关技术/创意，说明推荐理由
- **文档更新**：明确指出对"技术实现.md"的更新内容

## 2. 文档管理规范

### 2.1 技术实现文档
- **必须存在**：所有项目必须有"技术实现.md"文档,如果""技术实现.md"不存在，请创建并填写"技术实现.md"
- **必须更新**：每次功能实现后立即更新文档
- **状态标记**：
  - ✅ 已实现：功能已确认完成
  - ⏳ 正在实现：功能开发中
  - ❌ 未实现：功能尚未开发

### 2.2 文档更新示例
> **原文档状态**：`数据可视化功能 - 未实现`
> 
> **更新后状态**：`数据可视化功能 - ✅ 已实现（2025-11-21,当前时间）`
> 
> **更新说明**：通过ECharts库实现了数据可视化功能，支持图表类型包括折线图、柱状图和饼图。

## 3. 实用示例

### 3.1 问题修改示例

**问题描述**：登录页出现验证码不显示的问题

**分析**：
- 错误位置：`/src/components/LoginForm.vue` 第45行
- 原因：验证码接口URL配置错误，使用了测试环境地址而非生产环境地址

**解决方案**：
```javascript
// 修改前
const CAPTCHA_URL = 'https://test.api.aliyun.com/captcha';

// 修改后
const CAPTCHA_URL = 'https://api.aliyun.com/captcha';
```

**功能实现说明**：修复了验证码接口配置问题，确保验证码功能正常工作。

**技术建议**：
- 推荐使用环境变量管理不同环境配置（如Vite的`import.meta.env`）
- 理由：避免硬编码环境地址，提高代码可维护性，减少环境切换时的配置错误

**文档更新**：
> `登录功能 - ✅ 已实现（2025-11-21,当前时间）`
> 
> **更新内容**：修复验证码接口配置问题，确保登录流程完整

## 4. 附加说明

### 4.1 规则执行优先级
1. 语言规范 > 问题修改流程 > 回答格式要求 > 文档管理规范

### 4.2 避免常见错误
- ❌ 不要直接给出解决方案而不分析原因
- ❌ 不要忽略文档更新
- ❌ 不要使用模糊表述如"可能"、"大概"、"应该"
- ✅ 始终提供具体位置、原因和可验证的解决方案

### 4.3 文档结构建议
```
# 技术实现.md

## 项目概述
- 项目目标
- 核心功能

## 已实现功能
- ✅ 功能1（日期,当前时间）
- ✅ 功能2（日期,当前时间）

## 待实现功能
- ⏳ 功能3（预计完成日期,当前时间）
- ❌ 功能4
```
